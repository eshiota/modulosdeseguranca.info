Quase todos os bancos brasileiros implementam módulos de segurança, com a pretensão de garantir a segurança dos clientes em seus serviços online. A grande verdade é que este é um sistema falho, implementado em uma tecnologia com muitas brechas de segurança, que causa mais aborrecimentos do que realmente ajuda.

Vulnerabilidades

Os módulos de segurança de bancos como Banco do Brasil, Itaú e Santander são criados com uma tecnologia chamada Java Applets. O problema é que a mesma tecnologia que é usada pelos bancos para "garantir" a sua segurança pode justamente colocar você em risco.

A última falha de segurança do Java levou o Departamento de Segurança americano a recomendar que os usuários desativassem o Java imediatamente de seus navegadores. Veja o que o especialista em segurança Adam Gowdiak declarou sobre esta mesma falha:

Nós não ousamos dizer aos usuários que é seguro usar o Java de novo.

—Adam Gowdiak, Security Explorations

Um outro especialista declarou que a coisa mais segura a se fazer é assumir que o Java sempre será vulnerável.

Nos últimos 24 meses a tecnologia Java teve 16 atualizacões críticas e de segurança. Outras falhas de segurança podem levar até 2 anos para serem corrigidas.

Mesmo com todas essas recomendações, os bancos insistem em dizer que é uma medida de segurança.

Aborrecimentos

Como se não fosse suficiente os bancos exigirem que você use uma tecnologia que pode comprometer a sua segurança, ela também trás uma série de aborrecimentos aos clientes.

Instalar e configurar o seu navegador com o Java não é tarefa simples, mesmo para os mais experientes. Isso porque existem muitos fatores que podem fazer com que a instalação falhe, fazendo você perder tempo e dinheiro.

Além disso, por causa dos Java Applets, você não pode acessar os sites de bancos através de seu smartphone ou tablet. Não sem instalar um aplicativo específico para o seu dispositivo. O mais curioso é que o acesso realizado através destes aplicativos é feito sem o uso de Java Applets.

O acesso feito por aplicativos para smartphones e tablets não usa a tecnologia Java e isso não faz com que seu acesso seja menos seguro (pelo contrário, dada as falhas de segurança dos Java Applets).

Se o uso de Java Applets fosse realmente essencial, acha que os bancos iriam criar aplicativos sem eles? Pense de novo!

Problemas com Java Applets

Por causa da última falha de segurança dos Java Applets, decidiu que seria mais seguro atualizar os computadores desativando completamente o Java.

Por causa disso, milhares de clientes não podem mais acessar os serviços online, perdendo mais tempo e mais dinheiro. Os bancos cobram mensalmente a cesta de serviços (cobrança abusiva, mesmo com o aval do governo) que inclui o acesso ao Internet Banking. Se você não consegue configurar o seu computador com Java Applets, seja porque não sabe como fazer isso, ou porque simplesmente não é possível, está sendo lesado.

Como se proteger?

Se usar Java Applets não aumenta a segurança enquanto você acessa o site de seu banco, o que você pode fazer? A verdade é que grande parte das fraudes acontecessem por descuido dos usuários.

Siga as dicas abaixo e estará muito mais seguro, não apenas no site de seu banco.

  • Nunca abra e-mails enviados por bancos. É muito fácil forjar o autor de um e-mail, então, para que arriscar? Isto se aplica a e-mails avisando sobre atualizações cadastrais de bancos e CPF/CNPJ ou irregularidades junto ao SPC/Serasa. Na dúvida, ligue para estes órgãos e solicite mais informações sobre o problema.
  • Não use a mesma senha em serviços diferentes. A maioria dos usuários usa a mesma combinação de e-mail e senha em serviços diferentes. Embora seja prático, isso deixa você mais exposto, caso algum desses serviços sofra algum tipo de invasão.
  • Use uma senha forte. Use caracteres especiais, acentos e evite palavras disponíveis em dicionários. Quanto mais difícil for a sua senha, melhor.
  • Não acesse o site de seu banco de computadores compartilhados. Sabemos que nem todos tem seu próprio computador e que muitos utilizam computadores à partir de lan-houses. O problema é que você não tem como garantir que o seu acesso não está sendo monitorado.
  • Mantenha seu computador sempre atualizado. Atualize sempre o seu computador. Estas atualizações contém correções de bugs e falhas de segurança.

O que posso fazer?

Pelo fato de instituições bancárias sempre estarem embaixo das asas do governo, na prática, não há muito o que fazer a não ser reclamar. Reclame muito.

Reclame com seu gerente. Envie mensagens na ouvidoria do banco que você é cliente. Avise amigos, parentes e vizinhos para que façam o mesmo. Quem sabe assim, com muito barulho, possamos mudar alguma coisa!
